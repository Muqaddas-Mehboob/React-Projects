## 🛠️ How to Run

```bash
git clone https://github.com/Muqaddas-Mehboob/React-Projects.git
cd React-Projects
````

> Now go into any project folder and run:

```bash
npm install
npm start
```

Make sure Node.js and npm are installed on your machine.
